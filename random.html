Пример работы Java Script кода для вывода случайных фраз

Здесь был реализован вывод случайных фраз при помощи не сложного скрипта. Фразы находяться в массиве откуда выбираються в случайном порядке. Рандомный порядок обеспечивает скрипт с методом Math floor(). Также,по такому же принцыпу работает случайный фон для body. Кнопка сделана на чистом CSS.
